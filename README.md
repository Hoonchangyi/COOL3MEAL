# COOL3MEAL 레시피 검색과 식재료관리를 도와드립니다!

## 프로젝트 기간

* 2022-06 ~ 2022-11.


### 주요기능
* 레시피검색 : 검색 하고자 하는 음식의 레시피를 냉장고에 있는 식재료로 검색, 키워드로 검색 할 수 있습니다.

* 식단  공유 : 유저간 식단을 고유 할 수 있습니다. 아침 점심 저녁 개개인의 태마별로 커스텀해 공유합니다!

* 나의 냉장고: 가지고 있는 식재료의 유통기한 및 수량 관리를 도웁니다! 가지고 있는 식재료로 만들 수 있는 레시피를 찾아 볼 수 있습니다!


### 사용한 기술 및 라이브러리

* BackEndTool: Spring framework

* FrontTool: React


### 깨달은점

* 기존에 진행했던 프로젝트와는 다르게 React를 처음 접하고 적용해 진행했던 프로젝트 입니다. 
  기본적인 React 지식부터 응용하는 방법까지 공부하게 된 프로젝트였습니다.

* Spring Boot와 React를 함께 사용하려면 해야하는 환경설정을 배우게 됐습니다.

* Axios 비동기 통신을 이용하여 GET,POST API를 호출하고 보내는 방식을 알게 되었습니다.

* 백엔드에서 React.js 라우터에 있는 주소로 주소를 임의 변경하면, 매핑되는 주소가 없으므로 404 오류가 생긴다. 이를 방지하기 위해 에러가 발생하면 프론트엔드에서 작성한 frontend/src/index.html을 전송한다는 것을 알게되었습니다.
